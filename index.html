같은 단어라도 맥락에 따라 의미가 조금씩 달라요. 세 가지 관점으로 간단히 정리하고, 아래에 H(p) 미니 데모도 넣었습니다.

① 정보 엔트로피 (Information entropy)

불확실성/평균 정보량 H = - Σ p_i · log₂ p_i (단위: bit). 동일확률일 때 최대.

  • 예) 동전: p(앞)=p(뒤)=0.5 → H=1 bit
  • 예) 치우친 동전: p=0.9 → H≈0.47 bit

② 열역학적 엔트로피 (Thermodynamic entropy)

가역 열교환의 적분 ΔS = ∫ δQ_rev/T (단위: J/K).

  • 자유팽창: 외부 일 없이 부피가 늘면 미시상태 수가 크게 증가 → S 증가
  • 제2법칙: 고립계에서는 ΔS ≥ 0

③ 통계적 엔트로피 (Statistical entropy)

S = k_B ln Ω. 가능 배치 수 Ω의 로그. 동일확률이면 H = S / (k_B ln 2).

  • 동전 N개의 앞면 k개 배치는 Ω = C(N,k)

미니 데모: H(p) 곡선

슬라이더로 p(왼쪽 점유/확률)를 조절하면 H(p)=-[p log₂ p + (1-p) log₂(1-p)]가 어떻게 바뀌는지 확인해 보세요.

p=0.50 H(p)=1.000 bit
